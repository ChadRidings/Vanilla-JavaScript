Array Patterns (VanillaJS)

Find the index and value of the largest number in an array
const arr = [19, 37, 234, 37, 769, 13, 38, 95, 43, 987, 2, 16];
const largestValue = Math.max(...arr);
const largestIndex = arr.indexOf(largestValue);
console.log({ value: largestValue, index: largestIndex });
Output: { value: 987, index: 9 }
Find the index and value of the smallest number in an array
const arr = [19, 37, 234, 37, 769, 13, 38, 95, 43, 987, 2, 16];
const smallestValue = Math.min(...arr);
const smallestIndex = arr.indexOf(smallestValue);
console.log({ value: smallestValue, index: smallestIndex });
Output: { value: 2, index: 10 }
Sort an array from the smallest value to the largest
use slice to create a new array before sorting (avoid mutating the original array)
const arr = [19, 37, 234, 37, 769, 13, 38, 95, 43, 987, 2, 16];
const sortedArr = arr.slice().sort((a, b) => a - b);
console.log(sortedArr);
console.log(arr);
Output: [2, 13, 16, 19, 37, 37, 38, 43, 95, 234, 769, 987]
Output: [19, 37, 234, 37, 769, 13, 38, 95, 43, 987, 2, 16]
Sort an array from the largest value to the smallest
use slice to create a new array before sorting (avoid mutating the original array)
const arr = [19, 37, 234, 37, 769, 13, 38, 95, 43, 987, 2, 16];
const sortedArr = arr.slice().sort((a, b) => b - a);
console.log(sortedArr);
console.log(arr);
Output: [987, 769, 234, 95, 43, 38, 37, 37, 19, 16, 13, 2]
Output: [19, 37, 234, 37, 769, 13, 38, 95, 43, 987, 2, 16]
Get the sum of all numbers in an array
const arr = [19, 37, 234, 37, 769, 13, 38, 95, 43, 987, 2, 16];
const longhandSum = arr.reduce((acc, current) => acc + current, 0);
const shorthandSum = arr.reduce((a, b) => a + b, 0);
console.log(longhandSum);
console.log(shorthandSum);
Output: 2389
Output: 2389
